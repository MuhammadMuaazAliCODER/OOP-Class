## 🧩 Core OOP Concepts Covered

| Concept | Description |
|---|---|
| **Classes & Objects** | Defining blueprints and creating instances |
| **Constructors & Destructors** | Default, parameterized, and copy constructors |
| **Encapsulation** | Private/public access, getters & setters |
| **Inheritance** | Single, multi-level, and hierarchical |
| **Polymorphism** | Function overloading, overriding, virtual functions |
| **Abstraction** | Abstract classes and pure virtual functions |
| **File Handling** | Persistent data storage with custom file formats |
| **Dynamic Memory** | Pointers, `new`, `delete`, memory management |
| **Operator Overloading** | Custom behavior for built-in operators |
| **Templates** | Generic programming with class & function templates |

## 🛠️ How to Run Any Project

All projects are written in standard C++17. To compile and run:

```bash
# Navigate into the project folder
cd 01_Auth-System

# Compile
g++ -std=c++17 -o output main.cpp

# Run
./output
```
